# Professional Email Rewrite Prompt

Use this prompt to turn rough notes into a polished professional email.

## Prompt

Rewrite the following message professionally. Keep the tone clear, respectful, and concise. Make sure the message sounds polished but still natural.

## Example Input

Hey, I need to know if this meeting can move because the room is already booked.

## Example Output

Hello,

I wanted to check whether there is any flexibility with the meeting date or time. The requested location is currently reserved, so I would be happy to help identify another available option if needed.

Best,
Kelli
