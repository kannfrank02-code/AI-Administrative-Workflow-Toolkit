# AI Administrative Workflow Toolkit

This toolkit shows how I use AI to improve administrative operations, executive support, communication, scheduling, candidate coordination, and documentation workflows.

## What This Includes

- Professional email rewrite prompts
- Executive summary prompts
- Meeting agenda prompts
- Interview feedback summary prompts
- Travel request checklist
- Vendor request workflow
- ATS resume and cover letter prompt

## Tools Used

- ChatGPT
- Microsoft Word
- Excel
- Google Workspace
- Workday concepts
- Administrative operations experience

## Purpose

This project demonstrates how AI can save time, improve communication, organize information, and support better decision-making in administrative and operations roles.
